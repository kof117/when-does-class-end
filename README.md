# when does class end?
when does class end is a simple web app that will tell you when your class (or work day, or really anything) will end. input a start and end time, and you'll see the progress bar slowly fill up as your class progresses.

## features
percentage bar to help you visualize your progress through class
seconds counter to make the class feel shorter
auto fill correct start and end times (for a select few school districts because i am too lazy to implement support for all of them)
