# when does class end?
when does class end is a simple web app that will tell you when your class (or work day, or really anything) will end. input a start and end time, and you'll see the progress bar slowly fill up as your class progresses.

## features
- percentage bar to help you visualize your progress through the period
- seconds counter to make the period feel shorter
- auto fill start and end times (if your schedule happens to line up with what i've implemented)
